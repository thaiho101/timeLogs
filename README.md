# TimeLogs – Work Hour Tracker

## 🚀 Overview
A time tracking system built as a backup solution for unreliable workplace scanners, allowing users to log and verify work hours accurately.

## 🔥 Key Features
- Clock-in / clock-out tracking
- Automatic total hour calculation
- Record verification against official systems
- Admin dashboard for monitoring logs

## 📊 Impact
- Provided a reliable alternative to faulty workplace systems
- Ensured accurate time tracking and reduced discrepancies

## 🛠 Tech Stack
- PHP
- MySQL
- JavaScript
- HTML/CSS

## ⚙️ How It Works
Users manually log their working hours, and the system calculates totals automatically.  
Data can be reviewed and verified through an admin interface.

## 📌 Notes
Built based on real operational issues where existing time tracking systems were unreliable.
